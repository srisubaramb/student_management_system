Student Management System (MERN Stack Project)
Project Overview
Project Name
Student Management System
Technology Stack
●	MongoDB
●	Express.js
●	React.js
●	Node.js
Project Objective
The Student Management System is a simple web application that allows administrators to manage student records digitally. The system helps users add, view, update, and delete student information while learning the fundamentals of full-stack development using the MERN stack.
________________________________________
Project Scope
The application will allow administrators to:
●	Login to the system
●	Manage student records
●	Manage course information
●	Assign students to courses
●	View student details
●	Search students
●	Generate simple dashboard statistics
________________________________________

After completing this project, students will understand:

This project is suitable for beginners and can typically be completed within 1–2 weeks while covering all fundamental MERN Stack concepts.
